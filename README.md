# Titanic Gruppenprojekt

## Worum geht’s?
In diesem Repository befindet sich unsere Gruppenarbeit zur Analyse des
Titanic-Datensatzes im Rahmen des Moduls "Wissenschaftliches Arbeiten".
Der Datensatz wird mit R aufbereitet, analysiert und mit einfachen Grafiken
visualisiert. Außerdem nutzen wir Git und GitHub für die Zusammenarbeit
in der Gruppe.

## Datensatz
Der Titanic-Datensatz (`titanic.csv`) wird über Moodle bereitgestellt.
Im Projektverlauf erstellen wir daraus einen bereinigten Datensatz,
der für die Analyse verwendet wird.

## Vorgehen
1. Aufbereitung des Datensatzes (z.B. fehlende Werte, neue Variablen)
2. Erstellung von Funktionen für deskriptive Statistik und Visualisierung
3. Analyse des aufbereiteten Datensatzes
4. Zusammenfassung der Ergebnisse im Bericht (Latex)
